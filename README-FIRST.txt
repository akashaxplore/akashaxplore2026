❓ QUICK TROUBLESHOOTING
───────────────────────────────────────────────────────────────

Problem: Slides not moving
→ Check: event-slider.js is linked
→ Fix: Add <script src="event-slider.js"></script>

Problem: Countdown shows 00:00
→ Check: Date format is "DD Mon YYYY | HH:MM GMT"
→ Fix: Change to correct format, example: "01 Jan 2026 | 23:59 GMT"

Problem: Drag not working
→ Check: Try dragging on text, not on image
→ Fix: Image area prevents drag, drag on event name

Problem: Buttons not responding
→ Check: Open browser console (F12)
→ Fix: Look for error messages, check file links

Problem: Styles look broken
→ Check: CSS file linked BEFORE other stylesheets
→ Fix: Place <link rel="stylesheet" href="event-slider.css"> first
